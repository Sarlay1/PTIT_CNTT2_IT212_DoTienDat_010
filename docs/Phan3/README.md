# Hackathon IT212 - Ứng dụng AI trong Phát triển Phần mềm

## Giới thiệu

Repository này tổng hợp các bài thực hành của học phần Ứng dụng AI, tập trung vào việc sử dụng AI để phân tích, thiết kế và hỗ trợ phát triển phần mềm.

## Nội dung

### Phần 1

Tái cấu trúc `FarmMaintenanceService` theo nguyên tắc Open/Closed Principle bằng Strategy Pattern.

### Phần 2

Phân tích và xử lý lỗi Hibernate `TransientPropertyValueException`, đồng thời đề xuất cơ chế xử lý ngoại lệ bằng `@RestControllerAdvice`.

### Phần 3

Phân tích hệ thống AgriSmart Market, đề xuất Tech Stack, xác định Entity và thiết kế sơ đồ ERD bằng Mermaid.

## Cấu trúc thư mục

* `src/`: Chứa mã nguồn Java.
* `docs/`: Chứa báo cáo, lịch sử Prompt và tài liệu thiết kế.
* `prompts/`: Chứa các Prompt đã sử dụng trong quá trình làm việc với AI.

## Công nghệ sử dụng

* Java 17
* Spring Boot
* Spring Data JPA
* MySQL
* Mermaid
* Git

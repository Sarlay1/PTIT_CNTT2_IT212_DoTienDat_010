# Prompt sử dụng với AI - Phần 2

## Prompt 1

Phân tích đoạn mã Spring Data JPA sau và giải thích nguyên nhân gây ra lỗi:

`org.hibernate.TransientPropertyValueException`

Hãy giải thích dựa trên các trạng thái của Entity trong Hibernate (Transient, Managed, Detached).

## Prompt 2

Đề xuất các cách khắc phục lỗi trên. So sánh ưu điểm và nhược điểm giữa việc sử dụng `CascadeType.PERSIST`, `CascadeType.ALL` và việc lưu Entity liên quan bằng Repository trước.

## Prompt 3

## Prompt 4

Đề xuất cách sử dụng `@RestControllerAdvice` để xử lý các ngoại lệ của JPA hoặc `DataAccessException` nhằm trả về dữ liệu JSON thay vì HTTP 500 mặc định.
